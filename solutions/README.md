# solutions 폴더 가이드

이 폴더는 **언어별 풀이 구역**입니다.

## 언어별 경로
- `solutions/python/`
- `solutions/java/`
- `solutions/cpp/`
- `solutions/javascript/`

## 개인 작업 경로 권장
- `solutions/<language>/<이름>/weekNN_문제이름.확장자`

예시:
- `solutions/python/kimminsu/week01_두수의합.py`
- `solutions/java/parkjiyoon/week02_타겟넘버.java`

## 규칙
- 가능한 본인 언어 폴더 안에서만 수정
- 파일명은 주차 정보 + 문제명으로 작성
